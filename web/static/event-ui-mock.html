Key Design Decisions

  • Background color = classification status (single source of truth)
  • Skip button in bottom right corner - separate from project buttons
  • Skipped events have strikethrough text like declined meetings in Google Calendar
  • Skipped events have grayed calendar stripe to further de-emphasize
  • Project dot only shown for classified events (not pending)
  • Hover over project dot shows project name and confidence score (e.g., "SLSA (confidence: 65%)")
  • List view shows end time as well as start time
  • Consistent layout between calendar and list views
