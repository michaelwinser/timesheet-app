Design Notes

  • Invoice number format: {PROJECT}-{YEAR}-{SEQ} (e.g., ALPHA-2024-001). Project prefix from first word of project name.
  • Status workflow: Draft ‚Üí Finalized ‚Üí Paid. Can revert Finalized back to Draft.
  • Regenerate: Only available for Draft invoices. Re-queries time entries for the period.
  • Google Sheets: One spreadsheet per project (created on first export). Each invoice becomes a worksheet.
  • Re-export: Updates the existing worksheet in place. Last exported timestamp shown.
  • Invoice indicator: Small blue document icon on time entry cards. Tooltip shows invoice number.
  • Bill rate: Snapshot at invoice creation time. Regenerating uses original rate, not current project rate.
